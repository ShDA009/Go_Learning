# Go Learning — Сервис для изучения Go

Локальный сервис для индивидуального изучения языка Go по урокам с теорией, примерами кода и практическими заданиями.

## Быстрый старт

### 1. Установка зависимостей

```bash
go mod download
```

### 2. Импорт контента (первые 5 уроков)

```bash
go run ./cmd/ingest --db ./data.db --limit 5
```

### 3. Запуск сервера

```bash
go run ./cmd/server --db ./data.db --addr :8080
```

Откройте http://localhost:8080 в браузере.

## Структура проекта

```
cmd/
  server/      — веб-сервер
  ingest/      — CLI для импорта контента
internal/
  db/          — подключение к SQLite, миграции
  content/     — модели и репозиторий уроков
  ingest/      — crawler, parser, rewriter
  practice/    — runner и checker для проверки кода
  progress/    — отслеживание прогресса
  web/         — HTTP-обработчики и шаблоны
```

## Функции

- **Каталог уроков** — список модулей и уроков с навигацией
- **Страница урока** — теория, синтаксис, примеры, частые ошибки
- **Практика** — мини-задачи с автопроверкой через `go run` / `go test`
- **Прогресс** — отметки о прочтении и выполнении заданий
- **Заметки** — личные записи к каждому уроку
- **Поиск** — полнотекстовый поиск по урокам

## API

| Метод | Путь | Описание |
|-------|------|----------|
| GET | `/` | Главная страница (список уроков) |
| GET | `/lessons/{slug}` | Страница урока |
| GET | `/search?q=` | Поиск по урокам |
| POST | `/api/progress/lesson/{id}` | Обновить прогресс |
| POST | `/api/notes/lesson/{id}` | Сохранить заметку |
| POST | `/api/run` | Запустить Go-код |
| POST | `/api/check` | Проверить решение задачи |

## Лицензия

Для личного использования.
